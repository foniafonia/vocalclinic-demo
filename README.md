# VocalClinic Demo

Demostración interactiva de análisis vocal clínico.

## 🎯 Qué es

Demo funcional de herramientas de análisis vocal para logopedas.

## 🚀 Features

- Grabación de voz
- Análisis acústico básico
- Visualización de parámetros
- Reporte de resultados

## 🔧 Stack

HTML5 + JavaScript + Web Audio API

## 📋 Caso de uso

Demostración en jornadas, formaciones, webinars
